Reply code challenge 2019¶

A data science layman approach

A sample exercise, taken from contest's pdf (requires registration).

Basically: given a map representation and a list of customers coordinates, find the N most convenient places where build a base that satisfy those constraint:

  • cannot be built on mountains/customer cell
  • connect to at least one customer via a path made by NSEW (ie. no diagonal) moves

Paths costs are the sum of cells costs (defined below). Each reached customers grant a reward reward - moves_cost.

import networkx as nx

G = nx.Graph()
for x in range(len(map)):
    for y in range(len(map[0])):
        if map[x][y] != '#':
            G.add_node((x, y),
                       tile=map[x][y],
                       cost=tiles[map[x][y]]['cost'])

%%time

edges = []
for node in G.nodes:

    # possible NSEW neighbours' coordinates
    for x, y in [(node[0] - 1, node[1]),
                 (node[0] + 1, node[1]),
                 (node[0], node[1] + 1),
                 (node[0], node[1] - 1)]:

        try:
            # eclude mountains, negative coordinates
            if map[x][y] != '#' and not -1 in (x, y):
                edges.append((node, (x, y)))
        except IndexError:
            pass

G.add_edges_from(edges)

Find the sweet spots¶

A graphical view already suggest, at least seems so, where to put our bases in order to maximize profit. But where, exactly?

In the following code we check the reward for every customer starting from every possible location (ie. no mountains, no customer's cells):

In [9]:
%%time

def get_weight(u, v, data):
    return G.nodes[v]['cost']


distances = nx.all_pairs_dijkstra_path(G, weight=get_weight)

for node, distance in distances:
    for customer, cdata in customers.items():
        if customer not in distance:
            # todo
            continue

        path = distance[customer][1:]
        cost = 0
        for hop in path:
            cost += G.nodes[hop]['cost']
        reward = cdata['reward'] - cost
#         print(f'{node} -> {customer}: {reward} {path}')
        nx.set_node_attributes(G, {node:
                                   {customer: {'reward': reward,
                                               'path': path}}})
